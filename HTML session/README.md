# HTML session

Proyectos de práctica: páginas HTML estáticas usadas durante el curso.

Cómo verlos localmente:
1. Abrir el archivo `about.html` (u otro) en un navegador web.
2. O servir la carpeta con un servidor local (recomendado) como `Live Server` de VSCode o con Python:

   - PowerShell:
     python -m http.server 8000

Luego abrir http://localhost:8000 en tu navegador.

Qué contiene esta carpeta:
- `about.html`, `forms.html`, `table_demo.html`, etc.
